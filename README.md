# SplitWise Pro — Shared Expense Management

SplitWise Pro is a production-ready shared expense management application built to solve real-world problems around flatmate expenses. It includes advanced CSV anomaly detection to automatically ingest, fix, and flag messy historical data.

## Features

- **Authentication**: Stateless, secure JWT-based auth via Next.js Route Handlers and HttpOnly cookies.
- **Dynamic Membership**: Members can join or leave groups. The expense and balance engine gracefully handles active and inactive states.
- **Advanced CSV Import**: An intelligent parser and anomaly detector that scans for over 20 types of data inconsistencies (e.g., duplicates, incorrect split types, wrong currencies, negative amounts, incorrect dates).
- **Comprehensive Balance Engine**: Calculates exact debts with rounding protections using exact `Decimal` precision.
- **Debt Simplification**: An optimized debt resolution algorithm that reduces the total number of transactions needed for a group to settle up.
- **Premium UI**: Modern dark-mode interface built with custom Tailwind CSS, featuring glassmorphism cards, animated states, and full responsiveness.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Database**: PostgreSQL (via Neon)
- **ORM**: Prisma 7
- **Styling**: Tailwind CSS
- **Validation**: Zod
- **Authentication**: `jose` (JWT) + `bcryptjs`
- **CSV Parsing**: PapaParse

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm / yarn / bun

### Installation

1. Clone the repository and install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   Copy `.env.example` to `.env` (already done in this setup) and ensure your database connection string and JWT secret are set.
   ```
   DATABASE_URL="postgresql://user:password@host/db"
   JWT_SECRET="your-secret-key"
   ```

3. Generate the Prisma Client and migrate the database:
   ```bash
   npx prisma generate
   npx prisma migrate dev
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Using the Application

1. **Register**: Create a new account to get started.
2. **Dashboard**: View your total expenses, groups, and settlements at a glance.
3. **Groups**: Create a group and add members. Note that for the CSV import, users are automatically generated.
4. **Import CSV**: Navigate to the Import page, drop your CSV file, and watch the anomaly detection engine process the data.
5. **View Reports**: Review the parsed CSV import report, check exactly which rows were skipped or fixed, and analyze the anomalies.
6. **Balances & Settlements**: Check out a group's detail page to view net balances and use the one-click "Settle" feature generated by the debt simplification algorithm.
